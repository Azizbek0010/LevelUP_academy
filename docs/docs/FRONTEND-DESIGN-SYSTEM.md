# LevelUp Academy — Frontend Design System

> Дизайн-система выбрана пошагово: каждый токен отбирался из 4–5 живых вариантов.
> Базовый брендовый цвет — лайм `#C6FF34`. Тема — светлая (тёмная отклонена).

## Палитра (светлая тема — основная)

| Токен | Hex | Использование |
|---|---|---|
| `--bg` | `#F6FBEA` | Фон страницы (лайм-тинт) |
| `--surface` | `#FFFFFF` | Карточки — чисто белые, с явной тенью |
| `--text` | `#1D2417` | Основной текст — графитово-зелёный (одна семья с лаймом) |
| `--text-muted` | `#5E6E52` | Вторичный текст |
| `--border` | `#E6EDD8` | Границы, разделители таблиц |
| `--accent` | `#C6FF34` | Лайм: кнопки, прогресс-бары, бейджи, активная навигация |
| `--accent-ink` | `#141B10` | Текст НА лайме (тёмный, не белый) |
| `--sidebar` | `#1D2417` | Тёмный сайдбар/шапка (тот же графит, что текст) |
| `--sidebar-text` | `#DCE9CC` / dim `#8FA283` | Текст в сайдбаре |
| success | `#2ECC71` (bg `rgba(.14)`) | Статус «оплачен / активен» |
| danger | `#E8543E` (bg `rgba(.12)`) | «Должник / просрочка» |

- **Тень карточек:** `0 10px 28px rgba(29,36,23,.14), 0 2px 6px rgba(29,36,23,.08)`
- **Шкала оценок 0–100:** интенсивность лайма растёт с баллом (`rgba .35 → .55 → .78 → 1`)

## Типографика

- **Display (заголовки, цифры, логотип):** Manrope (variable 200–800), встраивается как woff2 data-URI
- **Body:** Segoe UI / system-ui
- **Цифры в таблицах:** `font-variant-numeric: tabular-nums`

## Компоненты (утверждённый макет дашборда v2)

- Сайдбар тёмный с иконками; активный пункт — лайм-подложка `rgba(198,255,52,.13)`
- Логотип-знак: кольцо прогресса (отсылка к 0–100)
- Стат-карточки: label + delta-бейдж (▲) + крупная цифра + спарклайн
- Таблица учеников: аватар-плитка, оценка + мини-бар, пилюли статусов
- График распределения оценок: 4 диапазона (0–59 / 60–74 / 75–89 / 90–100), тултипы
- Hover: карточки приподнимаются (`translateY(-2px)`), строки таблиц подсвечиваются `#F8FDF0`

## Реализация

Токены палитры ложатся в `themes.css` фронтенда (механика темизации — см. `FRONTEND-ARCHITECTURE.md`,
раздел «Темизация»; там `themes.css` уже приведён к этой лайм-палитре).
