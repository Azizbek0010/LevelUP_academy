# LevelUp Academy — Логотипы

Фирменные логотипы проекта в формате **SVG**. Шрифт **Manrope** (ExtraBold) вшит в каждый файл как data-URI — файлы самодостаточны, интернет и установленный шрифт не нужны, открываются в браузере и дизайн-редакторах.

Знак — незамкнутое кольцо-прогресс (отсылка к шкале оценок 0–100). Фирменный цвет — лайм `#C6FF34`, тёмный — графит `#1D2417`.

## Файлы

| Файл | Вариант | Назначение |
|---|---|---|
| `01-primary.svg` | Primary | Знак + «LevelUp Academy» в строку — основной логотип |
| `02-stacked.svg` | Stacked | Название в две строки (Level / Up Academy) |
| `03-vertical.svg` | Vertical | Знак сверху, надпись по центру |
| `04-reversed.svg` | Reversed | Надпись слева, знак справа |
| `05-twotone.svg` | Two-tone | «LevelUp» тёмный + «Academy» лаймовый |
| `06-mark.svg` | Mark | Только знак-кольцо (иконка/favicon) |
| `07-icon-ink.svg` | App icon (ink) | Тёмная плитка, лаймовый знак |
| `08-icon-lime.svg` | App icon (lime) | Лаймовая плитка, тёмный знак |
| `09-mono-ink.svg` | Mono ink | Монохром в один цвет (чернила) — для светлого фона |
| `10-mono-white.svg` | Mono white | Выворотка белым — для тёмного фона |

## Примечание

Текст в SVG — «живой» (не переведён в кривые), но нужный шрифт уже встроен в файл. Если конкретный инструмент не поддерживает встроенный в SVG `@font-face`, надпись подставит запасной шрифт — тогда нужна версия с текстом в кривых (outline).
